DataTables KeepConditions Plugin

Author: Justin Hyland
Links: GIT Repository | DataTables | Author | Other Projects by Author | Other Cool Stuff

Description: Store the DataTable conditions within the URL hash every time a condition is changed, such as the page, length, search or a column order, making it possible to copy/paste the URL. Once said URL is loaded, the conditions will be retrieved from the URL hash and implemented to the table

KeepConditions Examples

  • Basic Initialization - Enabling for Paging, Length, Search and Order
  • Advanced Initialization - Select specific conditions to keep conditions for
  • Basic Initialization w/ Button - Basic example with Copy URL Button
  • Multiple Table Initialization - Multiple Tables, Basic & Advanced w/ Button
  • Scroller & ColVis - Example of the compatibility with the Scroller extension and the ColVis button
  • Column Reordering - Example of the compatibility with theColReorder extension
